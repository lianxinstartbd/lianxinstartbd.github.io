06条件与判断

Posted on 2018-05-22 | In shell

1.字符串测试

=， ！=， -n，-z等：判断是否相等

必须要有 一个空格
不然会认为是一个普通的字符串，例如

\$a=\$b，输出0
实际应为：$a = $b，输出1.

Test与[是一样的作用。

2.整数测试

test num1 op num2

op：运算符，包括-eq，-gt，-lt，-ge，-le，-ne等。

3.文件测试

test op file

op：运算符，包括-a，-b，-c，-d等

主要是检测文件是否存在，文件类型以及文件的访问权限等。

4.逻辑操作符

与、或、非

-a -o ！

5.条件判断语句

if xxxx；then
xxxx
fi

#如果想将多条命令放在同一行中，则要使用；分开

:空命令，返回永远是0
Case语句

6.运算符

算术运算符、位运算符、自增资减

expr
\$((xxx))
\$[xxx]
let x+=10
